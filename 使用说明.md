# 备考宝典 APK 使用说明

## APK 位置

- 主资产：`build\out\exam-prep-handbook.apk`
- 兼容资产：`build\out\review-baodian.apk`
- 中文文件名：`build\out\备考宝典.apk`

## 功能概览

- 底部高频入口：`记题 / 刷题 / 错题 / 导图`
- 右上角侧边栏：课程选择、设置、建议
- 题目页支持左右滑动、进度条定位、错题本、当前题目导出
- 导图页支持画布拖动缩放、全展、收起、全屏
- 设置页支持主题、错题移除次数、导出提示词、版本更新
- App 打开且联网后会自动检查更新，也可以在设置页手动检查

## 构建校验口径

- App 显示名：`备考宝典`
- 当前课程：`计算机网络`
- Android 包名：`com.dz.networkquiz`
- provider authority：`com.dz.networkquiz.export`
- 默认更新源：`ZhiKong0/exam-prep-handbook-apk`

包名和 provider authority 是升级兼容点，不随英文项目名迁移。

## 联网更新

新版优先读取：

- `exam-prep-handbook-update.json`
- `exam-prep-handbook.apk`

过渡版本继续上传旧别名：

- `network_quiz_update.json`
- `review-baodian.apk`

这样旧版 App 仍能检查更新，新版 App 也会自动把旧仓库配置迁移到新仓库。
