# 自动发版说明

这个项目支持提交修复后自动发布 GitHub Release。

## 安装

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\tools\install_auto_release_hook.ps1 -RepoSlug ZhiKong0/exam-prep-handbook-apk
```

安装后会设置：

- `core.hooksPath=.githooks`
- `examprep.autoRelease=true`
- `examprep.releaseRepo=ZhiKong0/exam-prep-handbook-apk`

脚本仍会写入旧 `networkquiz.*` 配置作为过渡兼容。

## 行为

普通修复提交完成且工作区干净时，post-commit hook 会自动执行：

1. 递增版本号
2. 构建 `exam-prep-handbook.apk`
3. 生成 `exam-prep-handbook-update.json` 和旧别名 `network_quiz_update.json`
4. 创建 `Release prep: vX.Y.Z` 提交和 tag
5. 推送分支与 tag
6. 发布 GitHub Release，并上传新旧 APK 与元数据资产
7. 尝试把模拟器同步到最新版本

## 跳过自动发版

提交标题包含 `[skip-release]` 时会跳过自动发版。

```text
docs: tidy readme [skip-release]
```

## 日志

hook 日志写入：

```text
build/out/auto_release_hook.log
```
